# 🏢 Automated Rental Payment & SMS Notification System

A production-grade, live-deployable Property Management Dashboard and Tenant Checkout Portal. This system automates rent tracking, processes secure credit card payments, and dispatches real-time SMS receipts to tenants instantly upon payment.

## 📱 System Architecture & Interface Preview
* **Landlord Operations Dashboard:** View active tenant directories, review outstanding balances, register new units, and access individual tenant payment checkout portals.
* **Tenant Secure Checkout:** A clean, mobile-responsive invoice settlement interface for tenants to input card details and clear balances.
* **Automated SMS Ledger:** Leverages Twilio to fire global network cellular alerts instantly upon receipt of funds.
